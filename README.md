# parsnips
An interactive NLP sandbox.


### Currently supports:
<ul>
<li>Tokenization</li>
<li>Frequency Distribution</li>
<li>Named Entity Recognition</li>
<li>Lemmatization</li>
<li>Part of Speech Tagging</li>
<li>Constituency Parsing</li>
<li>Dependency Parsing</li>
<li>Coreference Resolution</li>
</ul>
